Sorting Algorithms: Simple Describtion and Python Approach

  • Average-/Best-/Worst-Case Time Complexity: the algorithm time requirement for respectively on averaged, at least, and at most input.

  • Space Complexity: the amount of working storage for an algorithm needs.

  • In-/Out-Place: Whether needing additional RAM space.

  • Stability: Whether maintaining the origianl order for same elements.

1. Bubble Sort

  1. Sort adjacent two elements.

  2. Repeat Step 1 for all elements. The Biggest one will appear at the last place.

  3. Repeat Step 1 and 2 for all elements but the last one. Each sorting array will ‘bubble’ the maximum one at the last place.

  4. When only ONE elements left, the sorting is complete.


def bubbleSort(arr):
    for i in range(1, len(arr)):
        for j in range(0, len(arr)-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

2. Selection Sort

  1. Check the minimum element for all unsorting array, swap it to the first place.

  2. Repeat Step 1 when no element left.


def selectionSort(arr):
    for i in range(len(arr) - 1):
        # Save the index of min
        minIndex = i
        for j in range(i + 1, len(arr)):
            if arr[j] < arr[minIndex]:
                minIndex = j
        # if i is not min, swap them
        if i != minIndex:
            arr[i], arr[minIndex] = arr[minIndex], arr[i]
    return arr

3. Insertion Sort

  1. Check the First element as sorted array, and the others as unsorted.

  2. Scan unsorted array, insert each element to its rightful place in sorted array.


def insertionSort(arr):
    for i in range(len(arr)):
        preIndex = i-1
        current = arr[i]
        while preIndex >= 0 and arr[preIndex] > current:
            arr[preIndex+1] = arr[preIndex]
            preIndex-=1
        arr[preIndex+1] = current
    return arr

4. Shell Sort

  1. Choose a Gap Sequence: $t_1, t_2, …, t_k$. Typically: $N/2, N/4, …, 1$, $N$ for the Length of input.

  2. According to the Length of Gap Sequence $k$, sort the array $k$ times.

  3. For each sorting, according to Gap $t_i$, split the array to $t_i$ subarrays and sort each one using Insertion Sort.

  4. Only when $t_i = 1$, sort the whole array and complete the algorithm.


def shellSort(arr):
    import math
    gap=1
    while(gap < len(arr)/3):
        gap = gap*3+1
    while gap > 0:
        for i in range(gap,len(arr)):
            temp = arr[i]
            j = i-gap
            while j >=0 and arr[j] > temp:
                arr[j+gap]=arr[j]
                j-=gap
            arr[j+gap] = temp
        gap = math.floor(gap/3)
    return arr

5. Merge Sort

  1. Sort every two elements. Now the every two elements are called Sorted Sequences.

  2. Allocate RAM with space of the sum of two Sorted Sequences.

  3. Set two points at the first place of each Sorted Sequences.

  4. Compare two pointed elements. The less one save into allocation RAM, and point to next elements.

  5. Repeat Step 4 when one point out of space. Then copy all the elements left in another sequence. Now the new sequence is a new Sorted Sequence.

  6. Repeat Step 2-5 when the Sorted Sequence is as long as the origianl.


def mergeSort(arr):
    import math
    if(len(arr)<2):
        return arr
    middle = math.floor(len(arr)/2)
    left, right = arr[0:middle], arr[middle:]
    return merge(mergeSort(left), mergeSort(right))

def merge(left,right):
    result = []
    while left and right:
        if left[0] <= right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    while left:
        result.append(left.pop(0))
    while right:
        result.append(right.pop(0))
    return result

6. Quick Sort

  1. Choose one element, called Pivot. Typically the first one.

  2. Set two points at first and last elements: $i, j$.

  3. $j$ start from the last and move to the first, finding the first element less than Pivot; $i$ start from the first and move to the last, find the first element greater than Pivot. When they both do, swap elements and keep moving and swapping.

  4. When $i = j$, swap Pivot and current element. Now Pivot is set the right order, and its left is all less than it and right greater.

  5. Repeat Step 1-4 until no swapping happens.


def quickSort(arr, left=None, right=None):
    left = 0 if not isinstance(left,(int, float)) else left
    right = len(arr)-1 if not isinstance(right,(int, float)) else right
    if left < right:
        partitionIndex = partition(arr, left, right)
        quickSort(arr, left, partitionIndex-1)
        quickSort(arr, partitionIndex+1, right)
    return arr

def partition(arr, left, right):
    pivot = left
    index = pivot+1
    i = index
    while  i <= right:
        if arr[i] < arr[pivot]:
            swap(arr, i, index)
            index+=1
        i+=1
    swap(arr,pivot,index-1)
    return index-1

def swap(arr, i, j):
    arr[i], arr[j] = arr[j], arr[i]

7. Heap Sort

  1. Build Heap with original array.

  2. Modify the Heap into a Max Heap, and swap the Root and end element. Remove the end element from the Heap.

  3. Now, the end element is the biggest one and in right order.

  4. Repeat Step 2 until all elements are set ordered.


def buildMaxHeap(arr):
    import math
    for i in range(math.floor(len(arr)/2),-1,-1):
        heapify(arr,i)

def heapify(arr, i):
    left = 2*i+1
    right = 2*i+2
    largest = i
    if left < arrLen and arr[left] > arr[largest]:
        largest = left
    if right < arrLen and arr[right] > arr[largest]:
        largest = right

    if largest != i:
        swap(arr, i, largest)
        heapify(arr, largest)

def swap(arr, i, j):
    arr[i], arr[j] = arr[j], arr[i]

def heapSort(arr):
    global arrLen
    arrLen = len(arr)
    buildMaxHeap(arr)
    for i in range(len(arr)-1,0,-1):
        swap(arr,0,i)
        arrLen -=1
        heapify(arr, 0)
    return arr

8. Count Sort

  • Prerequist: Input array must be Ranged Integer.
  1. Find the range (min, max)

  2. Count every element occurrences C, length is max - min

  3. For every C[i], fill the Result Array with value for index(C[i])-min, and count for C[i]


def countingSort(arr, maxValue):
    bucketLen = maxValue+1
    bucket = [0]*bucketLen
    sortedIndex =0
    arrLen = len(arr)
    for i in range(arrLen):
        if not bucket[arr[i]]:
            bucket[arr[i]]=0
        bucket[arr[i]]+=1
    for j in range(bucketLen):
        while bucket[j]>0:
            arr[sortedIndex] = j
            sortedIndex+=1
            bucket[j]-=1
    return arr

9. Bucket Sort

  1. Find the input range (min, max)

  2. Set Bucket number K, compute bucket range BR = (max-min)/K

  3. Traverse original array. For each element, put into right bucket.

  4. Sort in each bucket.

  5. Merge all bucket and output the sorted array.


def bucket_sort(s):
    min_num = min(s)
    max_num = max(s)

    bucket_range = (max_num-min_num) / len(s)
    # Create Bucket
    count_list = [ [] for i in range(len(s) + 1)]
    # Put element into Buckets
    for i in s:
        count_list[int((i-min_num)//bucket_range)].append(i)
    s.clear()
    # Regenerate sorted array
    for i in count_list:
        for j in sorted(i):
            s.append(j)

10. Radix Sort

  1. Unify all elements length (add 0s to the shorter elements).

  2. Start from the Least Significant Digit to the Highest, sort all by Stable sorting algorithm time by time.

  3. When the HSD sort is done, the array is set sorted.


def RadixSort(list):
    i = 0
    n = 1
    max_num = max(list)
    while max_num > 10**n:
        n += 1
    while i < n:
        bucket = {}
        for x in range(10):
            bucket.setdefault(x, [])
        for x in list:
            radix =int((x / (10**i)) % 10)
            bucket[radix].append(x)
        j = 0
        for k in range(10):
            if len(bucket[k]) != 0:
                for y in bucket[k]:
                    list[j] = y
                    j += 1
        i += 1
    return  list
